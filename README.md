# Cybersecurity Awareness Chatbot

##  Student Info
Name: [ANGEL MAKUA ST10472911]

##  Project Description
This is a console-based chatbot built in C# that helps users learn about cybersecurity practices such as password safety, phishing, and safe browsing.

##  Features
- Voice greeting using WAV file
- ASCII logo display
- User interaction with name input
- Cybersecurity-related responses
- Input validation
- Styled console interface

##  Technologies Used
- C#
- .NET Console App
- GitHub Actions (CI)

##  How to Run
1. Open project in Visual Studio
2. Run the application
3. Interact with the chatbot
